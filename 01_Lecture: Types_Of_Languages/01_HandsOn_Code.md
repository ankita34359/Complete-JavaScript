## Simple Examples

### Statically Typed Language Example (C):

```c
#include <stdio.h>

int main() {
    int age = 25;           // Declaring a variable with a specific type
    age = "twenty-five";    // Error: Incompatible type assignment
    printf("Age: %d\n", age);
    return 0;
}
```
