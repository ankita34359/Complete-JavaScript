## Simple Examples

### Statically Typed Language Example (C):

```c
#include <stdio.h>

int main() {

    // Declaring a variable with a specific type
    int age = 25;

    // Error: Incompatible type assignment           
    age = "twenty-five";

    printf("Age: %d\n", age);
    return 0;
}
```
